## How To Run

1. Go to [openweathermap.org](https://openweathermap.org/) and register.
2. Copy generated API Key 
3. In Folder root create .env file and make variable: REACT_APP_OPEN_WEATHER_KEY= YOUR_KEY_HERE
4. In Console run: 
     - npm install 
     - npm start
